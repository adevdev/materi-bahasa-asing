☆ Ungkapan Dengan Kata "Tener" ☆

Kata kerja *tener* berarti "memiliki", tetapi ia punya makna lain yang digunakan mengungkapkan ungkapan tertentu.

⚙️ *Perubahan Kata*
Berikut ini perubahan kata kerja "tener" sesuai subjek (untuk waktu sekarang):

yo ............................... tengo
tú ............................... tienes
él/ella/usted ............... tiene
nosotros ............... tenemos
vosotros .................... tenéis
ellos/ellas/ustedes .. tienen

Selain bermakna "memiliki" / "mempunyai", kata *tener* digunakan pada ungkapan lain. Contoh:

☆ Mengungkapkan usia
• _Yo *tengo* 25 años._
= Saya *berumur* 25 tahun.

☆ Mengungkapkan perasaan fisik
• tener frío = kedinginan
• tener calor = kepanasan
• tener hambre = lapar
• tener sed = haus
• tener sueño = mengantuk
• tener dolor de ... = sakit ... (pada bagian tertentu)

☆ Ungkapan-Ungkapan Lainnya
• tener prisa = buru-buru
• tener miedo a/de ... = takut pada ...
• tener celos = cemburu
• tener confianza = percaya diri
• tener cuidado = berhati-hati
• tener razón = benar
• tener éxito = sukses
• tener la culpa = bersalah
• tener suerte = beruntung
• tener lugar = berlangsung
• tener ganas de ... = merasa ingin ...

⚠️ Ingat untuk mengubah bentuk *tener* sesuai subjek.

Contoh:
• _*Tengo* hambre._
= *Saya (merasa)* lapar.

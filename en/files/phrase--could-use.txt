⭐ *Phrase* ⭐
_Kumpulan Kata_

                💁🏻‍♂️
  *could use something*

▶️ *Makna:* Ketika seseorang sedang memerlukan atau menginginkan sesuatu.
• sedang membutuhkan

✅ *Contoh:*
☆ *I _could use some help_ putting these decorations up if you're not too busy.*
=   Aku _lagi butuh bantuan_ memasang dekorasi ini kalau kamu nggak terlalu sibuk.

☆ *Their website _could use an upgrade._*
=   Halaman web mereka _sedang memerlukan peningkatan._

☆ *You both look as if you _could use a drink._*
=   Kalian berdua kelihatannya seperti _lagi butuh_ minum.

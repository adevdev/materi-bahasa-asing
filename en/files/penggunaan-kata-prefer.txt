☆ Penggunaan Kata 'Prefer' ☆

🔍 [Makna *"Prefer"*]___________
= lebih memilih (lebih suka sesuatu daripada yang lain)

⚙️ [Rumus]____________________
A. Pelengkap
1) PREFER *SOMETHING*
》 *I prefer _tea._*
= Saya lebih pilih/suka _teh._

2) PREFER *TO DO*
》 *I prefer _to stay_ home.*
= Saya lebih memilih _tinggal_ di rumah.

3) PREFER *DOING*
》 *I prefer _watching_ TV.*
= Saya lebih suka _menonton_ TV.

4) WOULD PREFER *TO DO* (formal)
    WOULD PREFER *DOING* (inform.)
》 *I would prefer _to discuss_ this matter.*
= Saya lebih memilih untuk _mendiskusikan_ masalah ini.

5) WOULD PREFER *SOMEONE TO DO*
》 *I'd prefer _you to drive._*
= Saya lebih memilih _kamu yang mengemudi._

6) WOULD PREFER *IT IF SOMEONE DID*
》 *I'd prefer _it if you drove._*
= Saya lebih suka _kalau kamu yang mengemudi._

7) WOULD PREFER *THAT SOMEONE DID*
》 *I would prefer _that you drove._*
= Saya lebih suka _kalau kamu yang mengemudi._


B. Kata Pembanding
1) PREFER A *TO* B
    PREFER A *RATHER THAN* B
    PREFER A *OVER* B
    PREFER A *INSTEAD OF* B
》 *I prefer soccer _to_ basketball.*
= Saya lebih suka sepakbola _daripada_ bola basket.

2) PREFER TO DO A *RATHER THAN* DO B (tanpa to)
》 *I would prefer to talk to the manager _rather than_ complain all day.*
= Saya lebih pilih berbicara dengan manajer _daripada_ mengeluh seharian.

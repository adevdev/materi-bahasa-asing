⭐ *PERKENALAN DIRI* ⭐
____________________________

❇️ *I am ...* / *I'm ...*
Saya adalah ...

❇️ *My name is ...*
Nama saya adalah ...

❇️ *I was born on ...*
Saya lahir pada tanggal ...

❇️ *I'm ... years old.*
Saya berumur ... tahun.

❇️ *I'm from ...*
Saya dari ...

❇️ *I live in ...*
Saya tinggal di ...

❇️ *I'm a/an ...*
Saya adalah seorang ...
❇️ *I work as a/an ...*
Saya bekerja sebagai seorang ...
• a student = pelajar
• a doctor   = dokter
• a nurse    = perawat
• a police officer = polisi
• an engineer = insinyur
• an architect = arsitek
• an athlete = atlit
• a teacher = guru
• a businessman = pengusaha
• a businesswoman = pengusaha (wanita)
• an enterpreneur = pengusaha kreatif
• a technician = teknisi
• a mechanic = mekanik
• a farmer = petani

❇️ *I live with my ...*
Saya tinggal bersama ... -ku.

❇️ *My hobby is ...*
Hobi saya adalah ...

❇️ *I like ...*
Saya suka ...

❇️ *I don't like ...*
Saya tidak suka ...

❇️ *I'm interested in ...*
Saya tertarik dengan ...

❇️ *I'm married.*
Saya sudah menikah.

❇️ *I'm single.*
Saya masih single.

❇️ *I'm engaged.*
Saya sudah bertunangan.

❇️ *I'm divorced.*
Saya sudah bercerai.

❇️ *I have ... children.*
Saya punya ... orang anak.

❇️ *I have a ...*
Saya punya seekor ... (peliharaan)
• a cat = kucing
• a dog = anjing

❇️ *I've been learning English for ...*
Saya sudah belajar bahasa Inggris selama ...

❇️ *Nice to meet you.*
Senang bertemu denganmu.

📺 *Contoh:*
Hello! My name is Dimas. I'm 17 years old. I'm a student. I live in Bandung. Nice to meet you.
